PokéCloud: Missão na Nuvem — Projeto React + Vite + Tailwind (Tema PokéCloud)

COMO RODAR
1) Instale Node.js (inclui npm).
2) No terminal:
   cd pokecloud-app-tailwind
   npm install
   npm run dev
3) Abra o link (ex.: http://localhost:5173).

Scripts:
- npm run build  -> gera build de produção
- npm run preview -> serve o build para teste

Tema Tailwind:
- Cores personalizadas em tailwind.config.js (pokecloud, pcloud-sky, pcloud-sun, pcloud-rose, pcloud-emerald).
